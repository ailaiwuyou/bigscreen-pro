# BigScreen Pro 用户手册

## 简介

BigScreen Pro 是一款企业级大屏可视化构建工具，帮助用户快速创建数据可视化大屏。

---

## 快速开始

### 1. 注册账号

1. 访问 BigScreen Pro 首页
2. 点击"开始使用"
3. 填写注册信息:
   - 用户名
   - 邮箱
   - 密码
4. 点击"注册"

### 2. 创建第一个大屏

1. 登录后，进入仪表盘管理页面
2. 点击"新建仪表盘"
3. 输入仪表盘名称和描述
4. 点击"创建"

### 3. 编辑大屏

进入仪表盘编辑器后，您可以:

#### 添加组件
- 从左侧组件库拖拽组件到画布
- 支持的组件类型:
  - 文本
  - 图片
  - 柱状图
  - 折线图
  - 饼图
  - 指标卡片
  - 等等...

#### 编辑组件
- 点击画布上的组件选中
- 在右侧属性面板修改:
  - 位置和尺寸
  - 样式属性
  - 数据配置

#### 调整布局
- 拖拽组件调整位置
- 调整组件大小
- 使用网格对齐

### 4. 保存和预览

- 点击工具栏"保存"保存更改
- 点击"预览"查看效果
- 点击"发布"发布大屏

## 功能说明

### 仪表盘管理

| 功能 | 说明 |
|------|------|
| 创建 | 新建空白仪表盘 |
| 编辑 | 修改仪表盘内容 |
| 复制 | 复制已有仪表盘 |
| 删除 | 删除仪表盘 |
| 发布 | 发布仪表盘供查看 |
| 设为公开 | 公开访问链接 |

### 组件库

#### 基础组件
- **文本**: 显示文字内容
- **图片**: 展示图片
- **视频**: 嵌入视频
- **边框**: 装饰边框

#### 图表组件
- **柱状图**: 适合比较分类数据
- **折线图**: 适合展示趋势
- **饼图**: 适合展示占比
- **散点图**: 适合展示分布
- **雷达图**: 适合多维度对比
- **仪表盘**: 适合指标展示
- **热力图**: 适合密度展示
- **地图**: 适合地理数据

#### 指标组件
- **指标卡片**: 展示单个 KPI
- **数字滚动**: 数字动画效果

### 数据源管理

支持的数据源类型:

| 类型 | 说明 |
|------|------|
| MySQL | 关系型数据库 |
| PostgreSQL | 高级关系型数据库 |
| REST API | HTTP API 接口 |
| JSON | JSON 文件 |
| Excel | Excel 文件 |
| CSV | CSV 文件 |

#### 添加数据源

1. 进入数据源管理
2. 点击"添加数据源"
3. 选择数据源类型
4. 填写连接信息
5. 点击"测试连接"
6. 保存

## 最佳实践

### 1. 布局设计

- 使用 1920x1080 作为标准分辨率
- 保持适当的边距和间距
- 重要信息放在左上角
- 使用栅格系统对齐

### 2. 颜色搭配

- 主色调: 不超过 3 种
- 使用品牌色作为强调
- 保持图表颜色一致性
- 考虑色盲用户

### 3. 数据展示

- 选择合适的图表类型
- 避免信息过载
- 使用标题和标签
- 保持数据更新

## 常见问题

### Q: 如何发布大屏?

A: 在编辑器中点击"发布"按钮，设置公开访问即可。

### Q: 如何嵌入其他网站?

A: 使用 iframe 嵌入:
```html
<iframe src="https://your-domain.com/preview/xxx" width="1920" height="1080"></iframe>
```

### Q: 数据如何实时更新?

A: 配置数据源后，系统会自动定时刷新数据，也可使用 WebSocket 实现实时推送。

### Q: 如何分享大屏?

A: 在仪表盘列表页面，点击"分享"获取公开链接。
