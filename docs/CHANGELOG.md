## [2.0.0] - 2024-03-15

### Added
- **全新编辑器引擎**：基于 Canvas 的高性能渲染引擎，支持 10k+ 元素流畅编辑
- **智能对齐系统**：支持像素级对齐、智能吸附、多元素对齐分布
- **组件市场**：可扩展的组件生态系统，支持第三方组件注册
- **实时协作**：基于 OT 算法的多人实时协作编辑
- **版本管理**：完整的版本历史、分支管理、版本对比功能
- **AI 辅助设计**：智能配色推荐、自动布局优化

### Changed
- 重构前端架构，采用微前端架构设计
- 升级 Vue 3 到最新版本，优化响应式性能
- 迁移状态管理从 Vuex 到 Pinia
- 重构后端 API，采用 GraphQL + REST 混合模式

### Deprecated
- 旧版拖拽引擎（将在 3.0 中移除）
- 同步数据导出功能（建议使用新版异步导出）

### Security
- 增强 JWT Token 安全性，支持 Token 绑定设备
- 添加 CSRF 防护机制
- 实现 API 请求签名验证
- 增强 SQL 注入防护

## [1.5.0] - 2024-01-20

### Added
- **大屏模板市场**：提供 100+ 行业模板
- **数据源增强**：支持 ClickHouse、Doris 等 OLAP 数据库
- **地图组件升级**：支持 3D 地图、飞线图、热力图层
- **移动端适配**：支持大屏移动端预览和控制
- **数据预警**：支持阈值告警、异常检测

### Changed
- 优化图表渲染性能，提升 40%
- 改进画布交互体验
- 重构数据源管理界面

### Fixed
- 修复大数据量下表格组件卡顿问题
- 修复某些情况下画布缩放失效问题
- 修复 WebSocket 重连机制
